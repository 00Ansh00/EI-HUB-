# EI-HUB 🚀

A centralized learning platform designed for Electronics & Instrumentation Engineering (EIE) students, providing quick access to academic resources, technical materials, and subject-wise study content through a clean and searchable interface.

## 🌐 Live Website

https://00ansh00.github.io/EI-HUB-/

## 📖 Overview

EI-HUB was developed to solve a common problem faced by EIE students—important resources are often scattered across multiple websites, PDFs, and platforms. This project brings them together into a single, organized portal that enables students to access study materials efficiently.

## ✨ Features

* Subject-wise resource organization
* Fast client-side search functionality
* Clean and responsive user interface
* Easy navigation across multiple subjects
* Lightweight static architecture
* SEO-friendly structure
* GitHub Pages deployment

## 🛠️ Tech Stack

* HTML5
* CSS3
* Tailwind CSS
* JavaScript
* Lunr.js (Client-Side Search)
* JSON
* GitHub Pages

## 📂 Subjects Covered

* Analog Electronics
* Digital System Design
* Signals & Systems
* Electronic Devices & Circuits
* Communication Systems
* Mathematics
* And other core EIE subjects

## 🔍 Search Functionality

The platform uses Lunr.js along with a JSON-based search index to provide fast client-side searching without requiring any backend server or database.

## 🚀 Deployment

The project is hosted using GitHub Pages, allowing easy access and maintenance without additional hosting infrastructure.
